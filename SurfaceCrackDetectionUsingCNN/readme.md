## 🏠 Surface Crack Detection Using CNN
This is a Convolution Neural Network(CNN) based project,helps to identify whether a picture of a wall or surface has a Crack or No-Crack.
### Overview
  Developed an AI-Based Surface Crack Detection System using Convolutional Neural Networks (CNN) to identify cracks on industrial surfaces from image datasets.Implemented image preprocessing, feature extraction, and binary classification techniques for accurate defect detection. Trained and evaluated the model using TensorFlow and Keras with visualization of prediction results and performance metrics.

---
### Dataset Information

Source : [Surface Crack Detection DataSet Link ](https://www.kaggle.com/datasets/arunrk7/surface-crack-detection)
Total Images : 40,000
Divided into two classes Crack(Positive) and No-Crack(Negative)

### 🛠️ Installation & Setup
Install libraries tensorflow,keras,scikit-learn,matplotlib

### 🚀 Running the Project
*  First Train the model      :  python3 SurfaceCrackDetectionUsingCNN.py train
*  Test the pre-trained model :  python3 SurfaceCrackDetectionUsingCNN.py test
*  Single Image Testing       :  python3 SurfaceCrackDetectionSingleImageTest.py

### 📊 Visualizations
*  Confusion Matrix,Classification Report,Accuracy Curve,Loss Curve,Sample Images,Missclassified Images,best model are Saved in Output_Saved folder
